Features

Everything You Need

From live system monitoring to biometric locks — QuinxOS has it all.

🖥️

Live Dashboard

Full-screen real-time system monitor with CPU, RAM, battery, network, and top processes. Refreshes every 2 seconds.

🎨

15 Theme Presets

Cyber Midnight, Matrix Green, Gruvbox, Tokyo Night, Catppuccin, Nord, Synthwave, and 8 more handcrafted themes.

🧩

Plugin System

Drop .sh files in the plugins folder — auto-loaded on boot. 11 built-in plugins for weather, QR codes, passwords, and more.

🔐

Quinx Shield

Encrypted terminal lock with 3-attempt lockout, recovery keys, and standalone unlock script. Never get locked out.

🔍

Command Palette

Fuzzy-search across all 34 features, aliases, and plugins. Like VS Code, but for your terminal.

📊

Git Dashboard

Full repo info at a glance — branch, commits, changes, stashes, tags, recent log, and changed files.

📌

Persistent Header

System info bar survives the clear command. Always see time, IP, battery, RAM, and git branch.

👤

Profile System

Multiple profiles for different contexts — Work, Personal, Hacking — each with separate theme, aliases, and plugins.

☁️

Dotfiles Sync

Push and pull your configs to a private GitHub repo. Restore your setup on any device instantly.

🎨

Theme Builder

Interactive color picker with 19 fields. Create unlimited custom themes and preview them live.

📡

Termux API Hooks

Battery status, GPS location, clipboard, flashlight toggle, vibration — all from the menu.

⏱️

QuinxBench

Benchmark your device — CPU prime calculation, disk read/write speed, network latency. Star-rated scoring.

Themes

15 Handcrafted Color Schemes

From cyberpunk neon to nature-inspired calm. Plus, build your own with Theme Builder.
