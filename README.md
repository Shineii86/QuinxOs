## What is QuinxOS?

**QuinxOS** is an all-in-one menu-driven terminal customization suite for Termux. It transforms your default Termux shell into a polished, modern, and secure development environment with one-click setup.

### Highlights
- 🎨 **Cyber Midnight Theme** — sleek dark color scheme with neon accents
- 🔒 **Quinx Shield** — encrypted terminal lock with 3-attempt lockout
- ⚡ **One-Click Setup** — install everything in a single command
- 🧩 **Smart Zsh Config** — Oh My Zsh + syntax highlighting + autosuggestions
- 📊 **System Info Dashboard** — view system details from the menu
- 💾 **Backup & Restore** — save and load your configurations
- 🖥️ **Custom Boot Banner** — personalized ASCII art at startup

---

## Features

### 🔒 Quinx Shield (Security)
Lock your terminal with an encrypted access key. Failed attempts auto-terminate the session.

### 🎨 Cyber Midnight Color Scheme
A carefully crafted dark theme with neon cyan, magenta, and electric blue accents.

### ⚡ Zsh Plugins
Pre-configured with syntax highlighting and autosuggestions for a better shell experience.

### 📊 System Info
View architecture, kernel, memory, disk usage, and uptime directly from the menu.

### 💾 Backup & Restore
Save your current shell configuration and restore it anytime.

### 🖥️ Custom Banners
Set personalized ASCII art banners that display on every new terminal session.

## Installation

```bash
git clone https://github.com/Shineii86/QuinxOS.git
cd QuinxOS
bash install.sh
```

---

## Menu Options

| Option | Feature | Description |
|--------|---------|-------------|
| 01 | Core Setup | Install dependencies, fonts, Oh My Zsh |
| 02 | Zsh Config | Reset Zsh environment |
| 03 | Switch → Zsh | Set Zsh as default shell |
| 04 | Switch → Bash | Set Bash as default shell |
| 05 | Custom Banner | Set boot banner name |
| 06 | Custom Theme | Set shell prompt name |
| 07 | Zsh Plugins | Install syntax highlighting + autosuggestions |
| 08 | System Info | View system details |
| 09 | Backup/Restore | Save or load configurations |
| 10 | Quinx Shield | Activate terminal lock |
| 11 | Remove Lock | Deactivate Quinx Shield |
| 12 | Update | Pull latest QuinxOS version |
